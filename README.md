 Exploratory Data Analysis on India State GDP Distribution
 Project Overview

This project performs a comprehensive Exploratory Data Analysis (EDA) on the GDP distribution across Indian states. The goal is to understand economic disparities, identify growth patterns, analyze sectoral contributions, and uncover insights that highlight the economic landscape of India at the state level.

The analysis includes:

GDP comparison across states

Sector-wise breakdown (Agriculture, Industry, Services)

Year-over-year growth patterns

Regional economic disparities

Visualization-driven insights

Objectives

*Explore and clean the India state GDP dataset

*Understand the variation in GDP across states

*Identify high-performing and under-performing states

*Analyze contributions of major economic sectors

Tools & Technologies Used

Python

Pandas – data cleaning & manipulation

NumPy – numerical computation

Matplotlib / Seaborn / Plotly – visualizations

Jupyter Notebook – analysis environment

Generate meaningful visualizations to support insights

Highlight trends and potential factors affecting state GDP
